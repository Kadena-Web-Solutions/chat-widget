💬 Chat Widget — Test Embed

This page embeds the chat widget for local development testing. The widget should appear as a floating button in the bottom-right corner.

Embedding Snippet (production)

<script src="chat-widget.js" data-client="mkstucco" data-sitekey="0x4..." async></script>

Configuration

The widget reads data-client and data-sitekey attributes from the script tag (or from the custom element directly). For local testing, configure below and click "Reload Widget".
